# ESP32 PZEM Firebase Monitoring System

This repository contains the source code of an ESP32-based
electrical energy monitoring and control system using
PZEM-004T v3.0 and Firebase Realtime Database.

## System Features
- Voltage, current, power, energy, frequency, and power factor monitoring
- Real-time data transmission to Firebase Realtime Database
- Remote relay control
- Local data display using I2C LCD 16x2

## Hardware Used
- ESP32
- PZEM-004T v3.0
- 4-Channel Relay Module
- LCD I2C 16x2

## Security Notice
WiFi credentials and Firebase authentication keys are intentionally
not included in this repository for security and publication purposes.
